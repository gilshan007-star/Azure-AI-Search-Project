# Azure AI Search Project

This project is an intelligent search application built using Azure AI Search and FastAPI.

## Technologies Used
- Python
- FastAPI
- Azure AI Search
- HTML, CSS, JavaScript

## Features
- Natural language search
- Cloud search with local fallback
- Web-based interface

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   uvicorn main:app --reload
